Intrução a web

História da Web

A Web começou como um projeto (de Tim Berners-Lee) para que cientistas pudessem compartilhar informações facilmente. Ela cresceu e se tornou a rede global de sites, aplicativos e mídias que usamos todos os dias.

Clients (Clientes)

É o software ou dispositivo que pede a informação. O exemplo mais comum é o seu navegador (Chrome, Firefox) ou um aplicativo no seu celular.

Servers (Servidores)

É o computador que armazena e entrega a informação. Quando você acessa um site, seu cliente (navegador) faz um pedido a um servidor, que envia a página de volta.

Linguagens de Programação

São as ferramentas usadas para construir tudo. HTML, CSS e JavaScript são usados para criar o que você vê no cliente (front-end). Java, C#, Python , etc. são usados para fazer o servidor funcionar (back-end).

Os Blocos de Construção do HTML

Esses tópicos cobrem os conceitos essenciais para criar qualquer página web.

A Estrutura Básica do HTML é o esqueleto obrigatório de toda página. Ela começa com "!DOCTYPE html", seguido pela tag "html" que envolve tudo. Dentro dela, temos o "head" (para informações sobre a página, como o título) e o "body" (onde fica todo o conteúdo visível). O conteúdo dentro do "body" é construído usando Tags. Pense nelas como "etiquetas" que dizem ao navegador o que cada pedaço de conteúdo é. Por exemplo, a tag "p" define um parágrafo, enquanto "h1" define um título principal. Elas geralmente vêm em pares, com uma tag de abertura (ex: "p") e uma de fechamento (ex: "/p"). Para dar mais informações a essas tags, usamos Atributos das Tags. Eles são instruções extras colocadas dentro da tag de abertura. Por exemplo, na tag de link "a", o atributo href="url" diz ao navegador para qual site o link deve apontar. Outros atributos comuns são id, class (para estilização com CSS) e src (para imagens).

Entre as tags mais comuns estão as de Textos. Elas estruturam o conteúdo escrito e incluem as tags de cabeçalho (de "h1" a "h6", para títulos de diferentes importâncias) e a tag "p" (para parágrafos comuns). Exemplo tags de texto:

  • Título 4
  • Título 5
  • Título 6

    • Parágrafo

Para organizar informações, usamos as Listas Ordenadas e Não Ordenadas. Listas Ordenadas ("ol") criam listas numeradas (1, 2, 3...).

  1. Primeiro item
  2. Segundo item
  3. Terceiro item
Listas Não Ordenadas ("ul") criam listas com marcadores (•). Ambas usam a tag "li" (list item) para envolver cada item da lista.
  • Item A
  • Item B
  • Item C
    • Por fim, os Links são o que tornam a web "conectada". Eles são criados com a tag "a" (âncora), e seu atributo mais importante é o href (hyperlink reference), que define o destino para onde o usuário será levado ao clicar. exemplo: Site da Dio
